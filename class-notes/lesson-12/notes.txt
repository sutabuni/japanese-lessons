[ A forma て dos verbos ]

Uso básico (quando aparece sozinha) é para dizer ações consecutivas.
O último verbo quando for falar multiplos é colocar o último como ます.

く/し:
（来）くる -> きて
する　-> して

Ichidan:
える/いる -> (tirar o る) -> て

Godan:
É mais complicado, tem várias maneiras de conjulgar:
う・つ・る -> って
む・ぶ・ぬ -> んで
く -> いて
ぐ　-> いで
す　-> して

喋る　-> しゃべって
走る　-> はしって
知る -> 知って (saber)
帰る -> かえって (retornar)
行く　-> 行って

A EXCESSÃO MAIS BIZARRA:
着る -> きて (vestir)
切る -> きって (cortar)

Outro Uso (de muitos)
~てください
~ていただはませんか
