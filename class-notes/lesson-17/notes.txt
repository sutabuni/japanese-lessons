Comparações　（比べる）

Pergunta:
Qual (de 2) é o mais x?
＿ と ＿ とどちらが （adj） ですか。
＿のほうが （adj） です。

Qual (de 3) é o mais x?
＿と＿と＿とどれ一番 （adj） ですか。
＿が一番　（adj）　です

(Resposta) Os dois são x:
どちも （adj） です

* Sempre vai indicar o que é mais.
* Para tempo dá para trocar o どち e どれ por　いつ. Dá para trocar para outras coisas no geral dependendo do contexto.

---

x é o maior y do mundo:
（substantivo） は （referência）（の中） で 一番　（adj）　です

---

x é mais y do que z
（sujeito） は （referência） より （adj） です

* Para falar que x não é mais y do que z, dá para usar o adjetivo no negatívo, mas não soa super natural.

---

Vocábulario/Kanji

背が高い: Altura da pessoa
面白い: Interessante, mas pode significar engraçado também
より: É "o mais" de superioridade
便利: Prático, facil de usar, conveniente
頭がいい: Gíria/expressão para inteligente
多い: Vários (quantidade)
人が多い: Populoso
